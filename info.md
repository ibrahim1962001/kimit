## 🚀 ملخص التحول الكبير

تم تحويل المشروع من مجرد صفحة بسيطة لتحليل البيانات إلى منصة **Business Intelligence (BI)** متكاملة واحترافية تعمل بالذكاء الاصطناعي، أطلقنا عليها اسم **Kemet AI Studio**. المشروع الآن يعتمد على أحدث التقنيات وأقوى معايير تجربة المستخدم.

---

## 🛠️ البنية التحتية والتقنيات

- **الأساس:** React 19 + TypeScript + Vite لضمان أقصى سرعة وأداء.
- **محرك البيانات:** PapaParse (لـ CSV) و SheetJS (لـ Excel) لدعم أحجام ملفات كبيرة مع معالجة ذكية (Sampling).
- **الرسوم البيانية:** استخدام Apache ECharts لتقديم رسوم تفاعلية احترافية بمستوى منصات عالمية مثل Power BI.
- **التصميم:** واجهة عصرية Dark Mode تعتمد على الـ Custom CSS و Glassmorphism.

## ✨ الميزات والوظائف (التي تم إنجازها بالكامل)

### 1. 📊 التحليل والداشبورد (Dashboard)

- **مؤشر صحة البيانات (Data Health Score):** مقياس لحظي يقيس جودة البيانات بناءً على القيم مفقودة، التكرار، والقيم الشاذة.
- **نظام الفلاتر الحية (Live Slicers):** فلاتر ذكية تسمح بتغيير كل محتوى الداشبورد بناءً على اختيارات المستخدم للأعمدة النصية.
- **صانع المخططات المخصص (Custom Chart Builder):** ميزة فريدة تتيح للمستخدم صنع رسم بياني مخصص له باختيار المحاور (Axes) ونوع الرسم.
- **تحليل العلاقات (Correlations):** كشف الروابط القوية بين المتغيرات رقمياً (Pearson Correlation).

### 2. 🤖 الـ AI مستشار البيانات (AI Advisor)

- **محادثة ذكية:** محرك قوي يعتمد على Groq API للإجابة على أي أسئلة حول البيانات أو أسئلة عامة.
- **إدارة الجلسات (Chat Sessions):** يدعم فتح محادثات متعددة، حفظ التاريخ في تابات (Tabs)، وحذف الجلسات القديمة.
- **التحكم الصوتي (Voice Control):** زر ميكروفون مخصص للتحدث مع الـ AI بدلاً من الكتابة.
- **التنفيذ التلقائي:** الـ AI يمكنه اقتراح تعديلات (مثل حذف التكرارات) وتنفيذها بضغطة زر واحدة.

### 3. 🧹 تنقية البيانات والتحكم الحي (Excel Playground)

- **محرر البيانات التفاعلي:** جدول بيانات احترافي (بناءً على react-data-grid) يسمح بتعديل الخلايا يدوياً وتحديث الداشبورد فوراً.
- **كشف القيم الشاذة بالألوان (Anomaly Highlighting):** تمييز القيم غير الطبيعية (Outliers) باللون الأحمر داخل الجدول لتنبيه المستخدم.
- **التشفير والذكاء الأمين (Anonymization):** ميزة لتشفير البيانات الحساسة (إيميلات، هواتف) تلقائياً لحماية الخصوصية.
- **التنقية الآلية:** زر واحد لمسح التكرارات وملء القيم مفقودة (Nulls) بالوسيط الحسابي.

### 4. 📥 خدمات التصدير (Exporting)

- **تصدير إكسيل (.xlsx):** تحميل البيانات بعد التعديل والتنقية كملف إكسيل احترافي.
- **تصدير PDF:** توليد تقرير شامل للداشبورد والرسوم البيانية بصيغة PDF.
- **تصدير CSV/JSON:** خيارات مخصصة للمبرمجين والمحللين.

## 📂 هيكل المشروع الأساسي

- `src/App.tsx`: المتحكم الرئيسي ونظام التنقل.
- `src/pages/DashboardPage.tsx`: شاشة التحليل والفلاتر الفورية.
- `src/pages/CleaningPage.tsx`: بوابة التنقية وجودة البيانات.
- `src/components/ChatPanel.tsx`: مركز المحادثة وإدارة الجلسات.
- `src/components/EditorSidebar.tsx`: محرّر الإكسيل التفاعلي.
- `src/lib/dataUtils.ts`: محرك العمليات الحسابية والتحليل الإحصائي.
